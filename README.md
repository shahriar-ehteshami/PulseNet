# PulseNet

A distributed pulse oximetry monitoring platform built with ESP32-S3, MAX30102, and Flask. PulseNet enables real-time collection, storage, visualization, and analysis of heart rate and SpO₂ data from multiple sensor nodes through a centralized monitoring dashboard.

## Components

- ESP32-S3 Pulse Oximeter Nodes
- Local Data Server
- Flask Dashboard
- Real-Time Monitoring
- Signal Processing & AI Research
